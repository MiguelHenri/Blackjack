## 🎮 How to run

First, compile the code running the following:
```bash
$ make all
```

Then, run the server in a terminal:
```bash
$ ./server
```

For each player, run the following in a different terminal:
```bash
$ ./client
``` 
> Make sure you configured the constants in `config.h` correctly.
